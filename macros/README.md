# Macros

Preprocessor practice with `#define` and simple compile-time constants.

## Contents

- [`macros.h`](macros.h) — Full inventory of macros and constants covering:
  - Unit conversions (time, length, mass, angles, temperature)
  - Mathematical constants and trig values
  - Geometry formulas (area, perimeter, volume, distance)
  - Physics constants and formulas
  - Chemistry constants and molar masses
  - Debug/display helpers
  - Array utilities and numeric limits
  - Memory management macros

> Source: IFT1166 course notes (Hiver 2026)
