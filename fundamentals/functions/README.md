# Functions

## Contents

- [`methodes.h`](methodes.h) — Comprehensive function library (~1500 lines) covering:
  - Basic math (factorial, power, logarithm, exponential)
  - Number manipulation (rounding, truncation, random)
  - Statistics & probability (mean, median, std dev, correlation, regression)
  - Geometry (areas, perimeters, volumes, trig, distances, transformations)
  - Vector operations (add, subtract, dot product, norm)
  - String manipulation (case conversion, reverse, palindrome check)
  - Array operations (sum, product, merge, filter, deduplicate)
  - Sorting & searching (bubble, selection, insertion, linear, binary)
  - Combinatorics (combinations, permutations, Pascal's triangle, Stirling numbers)
  - Number theory (primes, GCD, LCM, Fibonacci, divisors)
  - Unit conversions (temperature, distance, mass, base conversions)
  - Validation & verification
  - Matrix operations (add, subtract, multiply, transpose)
  - Calculus (derivatives, integrals, ODE solvers, Taylor series)
  - Optimization (bisection, Newton-Raphson, golden section, gradient descent)
