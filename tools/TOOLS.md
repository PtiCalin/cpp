# Tools

Outils internes du repo pour accelerer l'apprentissage et la pratique.

## Sous-dossiers

- `templates/` — Squelettes de code reutilisables.
- `scripts/` — Scripts d'aide (build, test, format, check).
- `datasets/` — Donnees d'entree pour exercices et mini-projets.

## Migration vers la nouvelle structure

- Script: `scripts/migrate-workspace-to-new-structure.ps1`
- Guide: `scripts/MIGRATION.md`

But pedagogique: reduire la friction pour se concentrer sur les concepts.
