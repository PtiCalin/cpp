# ROADMAP

> Learning path for **IFT1166 — Programmation orientée objet en C++**
> Université de Montréal · Hiver 2026

---

## Phase 1 — Foundations `[in progress]`

Core language building blocks.

- [x] Variables, types, constants
- [x] Control flow (`if`, `switch`, loops)
- [x] Functions and parameter passing — see [`methodes.h`](fundamentals/functions/methodes.h)
- [x] Arrays and manual size tracking
- [x] Preprocessor macros and `#define` — see [`macros.h`](macros/macros.h)

## Phase 2 — Modular Programming `[in progress]`

Organizing code across files.

- [x] Header / source separation
- [x] Include guards
- [ ] Basic compilation unit understanding
- [ ] Makefile / build-script basics

## Phase 3 — Object-Oriented Programming `[in progress]`

Class design and encapsulation.

- [x] Class design basics — see [`classes.h`](oop/classes-basics/classes.h)
- [x] Constructors and destructors
- [x] Member functions and operator overloading
- [x] Inheritance and polymorphism
- [ ] Multi-file class projects (separate `.h` / `.cpp`)
- [ ] Composition vs. inheritance trade-offs

## Phase 4 — File I/O and Persistence `[in progress]`

Reading, writing, and manipulating files.

- [x] File reading (text, binary, CSV) — see [`file-handling.cpp`](file-handling/file-handling.cpp)
- [x] File writing (overwrite, append, binary)
- [x] Directory operations
- [x] Search, replace, and statistics

## Phase 5 — Applied Console Projects `[in progress]`

End-to-end programs integrating all previous phases.

- [x] Generic program skeleton — see [`squelette.cpp`](clean-examples/squelette.cpp)
- [x] Employee management system — see [`exemple-employes.cpp`](projects/employee-management/exemple-employes.cpp)
- [x] Library management system — see [`exemple-bibliotheque.cpp`](projects/library-system/exemple-bibliotheque.cpp)
- [ ] Additional menu-driven console projects
