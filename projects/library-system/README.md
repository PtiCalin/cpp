# Library System

A library network management example.

## Contents

- [`exemple-bibliotheque.cpp`](exemple-bibliotheque.cpp) — Full program demonstrating:
  - Library branches, sections, book catalog
  - Client management and loan tracking
  - Employee management per library
  - Search by genre, author, and client
  - Top-rated books ranking
  - Financial summaries (catalog value, unpaid fines, acquisitions)
  - Monthly statistics

> Requires a companion `exemple-bibliotheque-data.h` header with struct/data definitions.
> Source: IFT1166 course notes (Hiver 2026)
