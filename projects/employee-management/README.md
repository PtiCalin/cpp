# Employee Management

An employee management system for a fictional company (TechNova Solutions Inc.).

## Contents

- [`exemple-employes.cpp`](exemple-employes.cpp) — Full program demonstrating:
  - Department and employee data structures
  - Search by ID, name, department, contract type, education level
  - Project tracking and employee assignments
  - Performance evaluations and leave tracking
  - Statistics (age, salary, experience averages; gender distribution)
  - Top N rankings (performance, salary)
  - Add, deactivate, salary raise, department transfer operations
  - Sorting (by name, salary, age) and filtering
  - CSV and JSON export/import
  - Full financial summary

> Requires a companion `exemple-employes-data.h` header with struct/data definitions.
> Source: IFT1166 course notes (Hiver 2026)
