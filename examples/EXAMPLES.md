# Examples (C++)

Exemples annotes et executables, organises par theme.

## Sections

- `fondamentaux/` — Variables, conditions, boucles, I/O.
- `fonctions/` — Valeur/reference/pointeur, surcharge, recursion.
- `pointeurs-memoire/` — Adresses, `new/delete`, pieges classiques.
- `poo/` — Classes, objets, constructeurs, encapsulation.
- `heritage-polymorphisme/` — `virtual`, `override`, classes abstraites.
- `stl/` — Conteneurs, iterateurs, algorithmes.
- `fichiers/` — `ifstream`, `ofstream`, parsing simple.
- `modern-cpp/` — `auto`, range-for, `constexpr`, bonnes pratiques.

## Convention de fichier

Chaque exemple devrait contenir:
1. Un objectif d'apprentissage.
2. Un code minimal compilable.
3. Une section "A retenir" (2-3 points).
