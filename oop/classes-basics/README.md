# Classes — Basics

## Contents

- [`classes.h`](classes.h) — Large class inventory (~1300 lines) covering:
  - **Math**: `Matrice` with operator overloading
  - **Geometry 2D**: `Point`, `Rectangle`, `Cercle`, `Triangle`, `PolygoneRegulier`
  - **Geometry 3D**: `Point3D`, `Sphere`, `Cylindre`, `Cone`, `Pyramide` variants
  - **Physics**: `Vitesse`, `Acceleration`, `Force`, `Particule`, `SystemeParticules`
  - **Electromagnetism**: `LoiOhm`, `LoiCoulomb`, `ChampElectrique`, `ChampMagnetique`
  - **Oscillators & Optics**: `OscillateurHarmonique`, `PenduleSimple`, `Lentille`, `Prisme`
  - **Chemistry**: `Atome`, `Element`, `Molecule`, `ReactionChimique`, `Solution`, `AcideBase`
  - **Biology**: `Organite`, `Cellule`, `ADN`, `Population`
  - **CS**: `ListeChainee`, `Pile`, `File`, `ArbreBinaire`, `AlgorithmeTri`
  - **Everyday objects**: `CompteBancaire`, `Livre`, `Voiture`, `Personne`, etc.
  - **Instruments**: `Barometre`, `Thermometre`, `CalculatriceScientifique`, `Compilateur`
