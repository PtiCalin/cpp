## Comment utiliser ce parcours

1. **Lire** le `README.md` du module — concepts + tableaux de synthèse.
2. **Étudier** le ou les fichiers `.cpp` de démo — compilables et annotés.
3. **Pratiquer** dans `examples/` sur le même thème.
4. **S'exercer** dans `exercises/` avec les exercices notés.
5. **Vérifier** dans `exercises/solutions/`.
